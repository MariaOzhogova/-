🔐 CSRF PoC — Тестирование защиты

Откройте этот файл в том же браузере, где вы залогинены на http://localhost:8080

Результаты появятся здесь...

Тест 1: POST без CSRF-токена

Отправляем POST на /api/save-lab без токена. Ожидание: 403 Forbidden

Тест 2: POST с поддельным CSRF-токеном

Отправляем POST с фейковым токеном. Ожидание: 403 Forbidden

Тест 3: Автоматическая отправка формы (классическая CSRF-атака)

Имитируем скрытую форму, которая отправляется автоматически. Ожидание: 403 Forbidden

Тест 4: GET вместо POST

Пробуем отправить GET на POST-эндпоинт. Ожидание: 405 Method Not Allowed

Тест 5: Доступ к защищённой странице без логина

Проверяем, перенаправляет ли / на логин. Ожидание: 302 → /login
